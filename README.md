## React Tools List

This is a small React App intended to show how to consume an API using [Axios](https://github.com/axios/axios), 
and make use of different basic React concepts such as stateless and statefull components and react-router-dom.

**Note: this app fetches its data from https://rocketseat-node.herokuapp.com/api/products**

You can clone this repository, cd in and run:

```
yarn install
```
This will install all the dependencies. To run the app just:
```
yarn start
```
